# Геометрия регионов Казахстана

`kazakhstan-adm1.geojson` — упрощённая статическая геометрия первого административного уровня из набора [geoBoundaries](https://www.geoboundaries.org/api/current/gbOpen/KAZ/ADM1/), сборка `9469f09`.

- источник исходных границ: OpenStreetMap / Wambacher;
- год, указанный источником: 2017;
- лицензия: Open Data Commons Open Database License 1.0;
- файл используется только как геометрия и не содержит показателей Verimed;
- атрибуция и ограничения подробно зафиксированы в `docs/regional-monitoring.md`.

Геометрия содержит 16 объектов. Она не используется для автоматического угадывания региона: связь выполняется только по явному справочнику `frontend/lib/region-map.ts`.
