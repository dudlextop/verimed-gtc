# Методика финансовой значимости

Финансовая значимость отражает объём медицинских услуг, связанных с сигналами риска. Она рассчитывается для сигнала, организации, региона, типа отклонения и анализируемого периода.

Для сигнала берётся объединение текущей медицинской записи и связанных записей. Сумма рассчитывается только по уникальным идентификаторам записей. Если запись связана с несколькими сигналами, внутри агрегата периода, региона или организации она учитывается один раз.

Снимок содержит общую сумму услуг, сумму услуг с сигналами, сумму высокого и критического риска, суммы по статусам экспертной проверки, число уникальных записей и пациентов.

Финансовая значимость отражает объём медицинских услуг, связанных с выявленными сигналами риска. Окончательную оценку выполняет уполномоченный специалист.
