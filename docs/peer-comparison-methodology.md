# Методика сравнения сопоставимых организаций

Организация сравнивается с группой, сформированной существующим механизмом по региону, типу и специализации. Если группа мала, критерии последовательно расширяются; ограничение сохраняется в снимке и показывается пользователю.

Для каждого показателя сохраняются значение организации, медиана, межквартильный типичный диапазон, отклонение, позиция, размер группы и надёжность. Сравнение включает услуги на пациента, стоимость на пациента, медианную стоимость, повторные и дорогостоящие услуги, сигналы на 1 000 услуг, конец месяца, нагрузку врачей, финансовую значимость и долю высокого и критического риска.

Надёжность считается высокой для достаточной группы без расширения, средней для группы от четырёх организаций или расширенных критериев и низкой для малой расширенной выборки. Малый размер группы всегда сопровождается ограничением.
