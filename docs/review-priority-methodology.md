# Методика приоритета проверки

Приоритет проверки — отдельный рабочий показатель от 0 до 100. Он не заменяет оценку риска и не меняет правила обнаружения отклонений.

## Формула сигнала

- 40% — оценка риска;
- 25% — финансовая значимость;
- 15% — повторяемость;
- 10% — число затронутых обезличенных пациентов;
- 10% — история экспертных решений и отсутствие текущего решения.

Каждый фактор нормализуется на шкалу 0–100. Вклады округляются методом наибольших остатков, поэтому их сумма точно равна итоговому приоритету. Уровни: 0–29 — низкий, 30–59 — средний, 60–79 — высокий, 80–100 — критический.

Для организации используется та же прозрачная структура весов. Повторяемость приближённо измеряется количеством сигналов и длительностью периода, а число пациентов нормализуется относительно максимума текущей выборки. Изменение риска показывается отдельно по историческим снимкам и пока не имеет самостоятельного веса.

При одинаковых данных и запуске формируется одинаковый приоритет.

## История экспертных решений

При повторном появлении по тому же fingerprint ранее подтверждённый сигнал задаёт нормализованное значение 70 для существующего фактора истории, ранее неподтверждённый — 25, а отсутствие текущего решения — 100. Вес фактора остаётся 10%, поэтому история не может единолично изменить приоритет более чем на 10 пунктов. Новое увеличение риска и финансовой значимости сохраняет преобладающее влияние. Аналитический риск при этом не пересчитывается.
