# Методика экспертной обратной связи

Обратная связь хранится как связанная append-only запись `ExpertFeedback` у события решения.

Для сигнала доступны оценки полезности, качества объяснения, достаточности данных и корректности приоритета. Для повторяющейся модели дополнительно оцениваются корректность группировки и полезность графа связей. Комментарий необязателен.

Агрегаты строятся по последнему действующему событию каждого fingerprint и показывают число рассмотренных объектов, распределение решений и оценки качества. Доля подтверждения показывается только при достаточной выборке; текущий минимальный порог — пять рассмотренных сигналов.

Обратная связь не переобучает Isolation Forest, не изменяет правила, пороги, риск или метрики качества. Она является дополнительным контекстом для методической проверки и будущей калибровки после отдельного согласования.
