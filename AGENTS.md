# Verimed — правила репозитория

- Пользовательский интерфейс и выводимый пользователю текст API — только на русском языке.
- Не утверждать, что система доказала нарушение. Verimed формирует сигналы для экспертной оценки.
- Все аналитические значения поступают через API.
- Backend: типизация Python, Pydantic-схемы, бизнес-логика в `services/`.
- Frontend: строгий TypeScript, доступные интерактивные элементы, состояния загрузки, ошибки и отсутствия данных.
- Синтетические данные не должны содержать реальные персональные данные.
